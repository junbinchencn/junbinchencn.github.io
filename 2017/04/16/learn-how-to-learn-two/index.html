学习困难科目的实用思维方法之理解组块

本文为 Coursera 课程 如何学习：学习困难科目的实用思维方法课程 第二周学习笔记

组块是什么

从神经科学的角度来讲，组块是一个信息片段。组块的信息可以通过经常使用联系在一起，举个例子，比如吃饭使用筷子夹东西往嘴里送这个过程就是一个组块，这个组块的信息有包括如何用手拿筷子，如何用筷子夹食物，如何用筷子将食物送到嘴边，嘴巴如何张开并接收食物。组块的信息也可以通过实际意义联系在一起，举个例子，比如树的信息就是一个组块，这个组块的信息包括树的高度，树的叶子形状，树的树干分支等。

组块的作用

组块可以被理解成闪烁的神经网络，这个神经网络包含了系列关键的想法，一旦被触发，组块内的信息会源源不断的涌现。这个过程我们可以看成是写作过程中的文思泉涌。这个神经网络也可以包含一系列关键动作，比如篮球运动员的投篮动作，接球–>下蹲–>抬手–>瞄准–>弹跳–>投球。组块是调动记忆内容的捷径。篮球运动员有投篮想法的时候，执行投篮动作不必去思考如何做动作，而是一气呵成。这是因为组块被直接调入工作记忆，大脑指挥身体立即执行组块内的动作。

组块的构建

组块可以被扩大，也可以被复杂化，但是怎么构建组块呢？
构建组块有以下几个关键步骤：

  1. 进入专注模式
  2. 理解基本概念
  3. 练习内容
  4. 回忆内容

对以上的内容是不是有种似曾相识的感觉？没错，这个就是比较科学的学习方式。
其中第4个步骤是单纯的回忆。那什么是单纯的回忆呢？和我们平常的回忆有什么不同呢？单纯的回忆就是脱离资料，努力回想关键点。这个是促进组块化的最佳方式之一，可以帮助我们更好的理解资料。在学习这个资料的场所之外可以取得更好的学习效果，因为大脑下意识情况下，会将学习场所和学习的资料进行联系，如果在学习场所之外回忆学习资料，可以减少这个影响。这样会使记忆更加牢固，理解更加深刻，更容易调用这个记忆。

组块的迁移

我们经常会看到这么一种说法，即使是不同的领域，也有一些互通的内容，在另外一个领域的经验往往可以应用到其他领域。这个互通的概念其实就是组块的迁移。

你在某个领域掌握的组块 可以帮助你学习另一个领域的组块 两个领域可能具有惊人的共通性 交替学习 选出的各种不同概念 方法与技术 在同一段时间全部加以练习 组块非常重要 但并不意味着能提高灵活性。但是这个交替学习的方式可以培养跳跃性思维，培养创造性想法。比如同一天时间内，你花一部分时间学习文学，花一部分时间学习科学，如此交替学习。

组块的欺骗

  1. 你是否真的是在学习？ 我们在学习完一个材料，有时候觉得自己已经完全理解了这个材料了，觉得自己已经学完了这个材料。那么这个时候我们要千万小心，千万不要被自己的错觉欺骗了，误以为我们自己是在学习这个材料，误以为我们自己已经完全理解这个材料了。那要怎么证明自己已经理解了这个材料了，一个简单的方法是用一些小测验来证明自己是否真的掌握了这个材料，也可以采用回忆的方式。
  2. 你是否记住了标记的内容？ 在学习材料的过程中，会给一些材料的内容进行标记，这会让自己以为记住了那些内容，实际上并没有掌握。那么怎么避免这个缺陷呢？那就是尽量少做标记。

  3. 只练习简单的部分你就懂了？只练习简单的部分会给你造成一种错觉，就是你已经全部掌握了这个材料，有意练习困难的部分，可以让你更好的掌握材料。

  4. 没有犯错你就会了？学习过程中不要害怕犯错误，犯错误可以让自己察觉到自己对自己能力的评估错觉。

思维定势

思维定势，是对事物已有的看法 或者说是你已经充分发展巩固了的神经模式这会妨碍你产生更好的想法或者让你不能灵活地接受那些更好或更合适的新方法。那么如何尽量避免思维定势呢？一个比较好的方法是利用组块的迁移 交替学习不同学科的内容，培养创造性思维和跳跃性思维。给你的思维更多的思考方式。
