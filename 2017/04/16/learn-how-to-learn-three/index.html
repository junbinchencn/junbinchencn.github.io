学习困难科目的实用思维方法之对抗拖延

本文为 Coursera 课程 如何学习：学习困难科目的实用思维方法课程 第三周学习笔记

为什么喜欢拖延


当做一件我们不喜欢的事情的时候，会触发我们大脑的痛苦意识，这个时候我们下意识会转而去做令我们愉悦的事情，比如玩手机，玩游戏。但是这样做真的会令我们更加愉悦么？只能说这个是愉悦是暂时的，时间消逝之后，我们的痛苦会加倍，因为我们需要做的事情没有完成。拖延就像是温水煮青蛙，每天都拖延，每天的水都在慢慢加热，等到水开了之后，我们发现自己的拖延症已经晚期了。

有一种模式叫做僵尸模式

僵尸模式就是一种惯性，也就是习惯。比如刚开始学倒车的时候，你会觉得非常难。但是当你学会了之后，倒车这件事则变得非常简单。因为你开始学习倒车的时候相对紧张，你面对着各种各样的复杂信息，觉得这项任务变得异常艰巨。但是，一旦你学会了之后，也就是倒车这个行为在你的大脑里面形成了组块。你只需要触发这个倒车的想法，你的大脑就会进入僵尸模式，它只处理关键信息，不会因为要全面兼顾而力不从心。
习惯可以分为4个阶段。

  1. 第一个阶段是信号（cue），这个是进入僵尸模式的触发器。这个信号可能是一个微信消息，一个电话。信号没有好坏之分，重要的是我们接受信号之后的反应。
  2. 第二个阶段是惯性（routine），也就是僵尸模式，你的大脑在接受到信号之后做出的习惯性反应。
  3. 第三个阶段是奖励（reward）,任何一种习惯得以养成都是因为它们可以回报我们，它会立即带给我们愉悦感。拖延是极易产生的习惯，因为你将关注点转向了更令人愉悦的事情后所带来的奖励更加迅速而简单。但是好习惯也是有奖励的，想办法奖励好习惯有助于养成习惯。
  4. 第四个阶段是信念，坚信习惯，习惯就会强大起来。

关注过程而不是结果

当你开始学习新东西的时候总是会有一些负面情绪，甚至是你喜欢的东西也是会这样的。对于这些负面情绪，不同的处理方式会带来不同的效果。研究者们发现，不拖延的人会把负面情绪放到一边并对自己说，别浪费时间了，马上动工。 一旦你开始了，就会感觉好多了。注重过程而不是结果是一个非常好的方法。即学会把注意力集中在过程而不是结果上，过程就是指流动的时间，以及这段时间里的习惯和行动。比如 我要花60分钟来工作，而结果是工作产出。专注于过程的最简单方法就是把注意力集中在一个番茄工作周期，即25分钟的时间片段上，而不是完成某一项任务上。这里的基本思想在于你大脑中的僵尸习惯部分喜欢过程，因为它能够毫不费力的前进，要让友好的僵尸习惯来帮助过程的进行，这样做出结果就比较简单了。专注于过程而不是结果 ，你便不会总是去想还有多久才能做完，从而使自己在工作或者学习的过程中放松下来。

你想要对抗拖延的方法

  1. 写下本周重要任务列表。
  2. 在前一晚写下第二天的每日任务，研究表明，这可以让你在潜意识下设法解决任务，从而找到完成方法 。睡前写好任务清单，征召僵尸，帮助你完成第二天列表里的项目。 如果不把任务写下来 它们就会潜伏在四个工作记忆插槽的边缘，在寸土寸金的工作记忆区里尸位素餐，但将任务写到纸上，工作记忆就可以用来想别的事情了 。
  3. 将任务划分成小块任务，使用番茄工作法工作。确保完成任务后，你和你的僵尸可以得到奖励，完成任务后细细品味你的成功和喜悦，放大你的愉悦感。

4.时刻留意并记录触发你进入拖延模式的信号，尽量避免这些信号的干扰。比如手机消息会使你进入拖延模式，那么是否可以将你的手机收起来，或者不带在身上。

5.信任你的体系。在一个番茄时间内集中精力做好工作或者学习，在完成任务后毫无负罪感地尽情放松。

6.当你依旧拖延的时候，采用B计划，慢慢实践新体系。

7.每天先做你最不喜欢的工作。
